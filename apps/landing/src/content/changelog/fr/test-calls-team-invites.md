Nous avons ajouté deux fonctions pour préparer LobbyStack avant les vrais appels : les appels de test vers votre réceptionniste IA et les invitations de membres dans votre espace d'entreprise.

Vous pouvez maintenant appeler votre réceptionniste IA depuis LobbyStack. Lancez un appel en direct depuis le tableau de bord, écoutez l'accueil, vérifiez les infos d'entreprise qu'il utilise et ajustez les consignes avant de donner le numéro à vos clients.

Vous pouvez aussi inviter votre équipe dans le même espace. Ajoutez les personnes qui aident avec les appels, les réservations, la configuration ou la facturation, puis gardez le travail dans la même entreprise LobbyStack.
