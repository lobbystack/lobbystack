# LobbyStack Mintlify Docs

This directory is the Git-backed source for the public LobbyStack docs at `https://docs.lobbystack.com/`.

The initial purpose of this directory is to give Mintlify a repository path containing `docs.json` so the existing dashboard-managed docs can be connected and synced into Git.

Use this path in Mintlify Git settings:

```text
mintlify
```
