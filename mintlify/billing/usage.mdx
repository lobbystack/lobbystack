title: "Usage and limits"
description: "Track voice minutes, SMS segments, outbound call attempts, knowledge storage, and Pro overages."
---

The **Settings** > **Usage** page shows hosted usage for the current billing period. Usage resets at the start of each period.

## Usage metrics

| Metric | Meaning |
| --- | --- |
| Voice minutes | Time spent on live voice calls. |
| Alert SMS segments | SMS notification segments sent by the system. |
| Outbound calls | Outbound call attempts, such as live transfer attempts. |
| AI SMS segments | Metered SMS segments for AI SMS conversations when the add-on is active. |
| Knowledge storage | Stored knowledge content size. |

## Included usage

| Metric | Free | Pro |
| --- | --- | --- |
| Voice minutes | 10 min | 80 min |
| Alert SMS segments | 10 | 50 |
| Outbound calls | 2 attempts | 20 attempts |
| Knowledge storage | 100 MB | 2 GB |
| AI SMS segments | Upgrade to Pro add-on | Metered add-on |

## How limits work

Free hosted workspaces include a set usage pool for each billing period. When an included pool is used up, that usage type pauses until the next period or until the workspace upgrades.

Pro supports billable overages for voice, alert SMS, and outbound call attempts. Knowledge storage has a fixed plan allowance; remove old content or upgrade before adding more after the allowance is full.

## Pro overage rates

| Usage type | Rate |
| --- | --- |
| Voice | $0.18 per extra minute |
| Alert SMS | $0.02 per extra segment |
| Outbound calls | $0.02 per extra attempt |

AI SMS is billed separately through its add-on at $0.03 per segment.

## View usage

<Steps>
  <Step title="Open Settings">
    In the dashboard, open **Settings**.
  </Step>
  <Step title="Open Usage">
    Select **Usage** from the settings sidebar.
  </Step>
  <Step title="Review included usage">
    Check current usage, remaining quota, usage status, and reset timing.
  </Step>
  <Step title="Review pay-as-you-go usage">
    On Pro, review estimated overage counts and cost for the current period.
  </Step>
</Steps>

<Note>
  Displayed overage amounts are estimates for the current period until the final invoice is created.
</Note>
